* Learn Python and get comfortable calling LLM APIs
* Think in systems, not prompts, and build small tools that solve problems
* Practice programming in Python and resolve small 'toy' problems on sites like Codewars
* Deploy projects on a real cloud, add auth, billing, and health checks
* Aim to ship 3+ small AI apps and try to sell them
