* El agente de revisión de código se asegura de que los problemas de seguridad se resuelvan antes de que se mezcle el código.
* El agente de revisión de código se asegura de que los problemas de búsqueda de bolsas se resuelvan antes de que se mezcle el código.
* El agente de revisión de código se puede configurar para trabajar con Cloud Code.
* El agente de revisión de código es una de las herramientas que se pueden utilizar para mejorar la calidad del código.
