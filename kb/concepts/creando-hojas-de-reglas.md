* Las hojas de reglas son un conjunto de reglas que siempre deben seguirse.
* Las hojas de reglas se pueden utilizar para asegurarse de que el código se ejecute correctamente.
* Las hojas de reglas se pueden configurar para trabajar con Cloud Code.
* Las hojas de reglas son una forma de garantizar la calidad del código.
