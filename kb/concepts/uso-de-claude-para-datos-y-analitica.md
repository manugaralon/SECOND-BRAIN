* La CLI 'bq' puede ser utilizada para extraer y analizar métricas de inmediato.
* La habilidad de BigQuery integrada en el código base puede ser usada para consultas analíticas directas.
* Claude puede ser utilizado con cualquier base de datos que tenga una CLI, MCP o API.
