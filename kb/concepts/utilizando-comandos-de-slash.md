* Los comandos de slash son cortos de customizaciones que se pueden utilizar para realizar tareas específicas.
* Los comandos de slash se pueden utilizar para leer información sin tener que reescribir nada.
* Los comandos de slash se pueden configurar para trabajar con Cloud Code.
* Los comandos de slash son una forma de automatizar tareas en Cloud Code.
