* Las habilidades en Cloud Code son un conjunto de instrucciones que se siguen cuando se reconoce una situación específica.
* Las habilidades en Cloud Code se pueden utilizar para realizar tareas específicas.
* Las habilidades en Cloud Code se pueden configurar para trabajar con Cloud Code.
* Las habilidades en Cloud Code son una forma de automatizar tareas en Cloud Code.
